 AI Portfolio – Ufhano Tshivhidzo

An AI-powered portfolio application that allows recruiters and visitors to chat with an intelligent assistant to learn about **Ufhano Tshivhidzo’s skills, experience, projects, and DevOps background**.

Built as a full MERN stack application with a modern chat UI and deployed to production.

 Live Demo

Frontend (Vercel):
   https://ai-portfolio-ivory-seven.vercel.app

Backend API (Render):
  https://ufhano-gpt.onrender.com

 Features

- GPT-powered chat interface
-  AI assistant trained to answer recruiter-style questions
-  Real-time chat experience with loading states
-  Secure backend with environment variables
-  Fully deployed frontend & backend
-  Responsive, modern UI

Running Locally
1️⃣ Clone the repository
git clone https://github.com/Ufhano/Ai-Portfolio.git
cd Ai-Portfolio

2️⃣ Start the backend
cd server
npm install
npm run dev

3️⃣ Start the frontend
cd client
npm install
npm run dev


Frontend will run on:

http://localhost:5173
